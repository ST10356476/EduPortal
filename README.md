# Student Management App

**Student Management App** is a desktop application designed to simplify the management of student data. The app provides features such as student record creation, updates, deletion, and retrieval, making it a valuable tool for academic institutions or organizations managing student databases.

---

## Features
- **CRUD Operations**: Add, edit, delete, and view student records.
- **Search Functionality**: Quickly search for students by ID, name, or other attributes.
- **User-Friendly Interface**: Intuitive design for easy navigation and operation.
- **Data Integrity**: Ensures secure and accurate storage of student information.

---

## Technologies Used
- **Programming Language**: Java
- **Build Tool**: Apache Ant (`build.xml`)
- **IDE**: Developed using NetBeans
- **Testing Framework**: JUnit for unit testing
- **Version Control**: Git

---

## Installation
### Prerequisites
- Java Development Kit (JDK) 8 or later
- Apache Ant for building the project
- NetBeans (optional, for development)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/ST10356476/Student-Management-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Student-Management-App
   ```
3. Build the project using Apache Ant:
   ```bash
   ant build
   ```
4. Run the application:
   ```bash
   ant run
   ```

---

## Project Structure
- **build**: Compiled project files.
- **nbproject**: NetBeans project files for configuration and IDE support.
- **src**: Source code for the application.
- **test/abcstudentmanagementapp**: Unit tests for the app's features.
- **testresources**: Resources used for testing, including mock data.

---

## Usage
1. Launch the application.
2. Use the menu options to:
   - Add a new student record.
   - Edit an existing record.
   - Delete a record.
   - Search and view student details.
3. Use the search bar to find students efficiently.
